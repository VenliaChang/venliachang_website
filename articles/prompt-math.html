用數學思維，精準下 Prompt Precision Prompting: A Mathematical Approach to Language Models

Venlia Chang
2025
3 分鐘閱讀

每次下 Prompt 的瞬間，你的腦袋在轉什麼？

怎麼問 AI 才聽得懂？聽說對它強硬一點，準確率更高？有禮貌的「請」，反而讓回答品質打折？

身為數學背景出身，我的直覺是——先問本質。

LLM 的本質是什麼？

文字向量化、向量相似性、文字接龍的條件機率、時間序列、立體的統計權數。白話文：在高維空間裡，看兩個詞的「方向」有多接近。

第一個 Prompt，是 AI 的定錨點

第一句話決定了搜尋空間的起始座標。我的習慣是：先交代背景與輪廓，讓模型校準定位，再進入核心問題。

隨著對話推進，向量不斷疊加，問答的語意空間也逐步擴展。距離當下越近的對話，模型賦予的記憶權重越高——就像昨天的晚餐，永遠比三年前的某頓飯記得更清楚。

為什麼「請」反讓準確率下降？

因為模型的核心是向量相似性，而非禮貌判斷。「請」這個字語意資訊量極低，在問題的向量方向上只是多了一個噪點，稀釋了你真正想問的語意密度。

理解了底層邏輯，上述現象自然迎刃而解。不過，習慣難改，我還是繼續留著這個噪點。🤗

When you send a prompt to a language model, what's actually going through your mind?

How should you phrase things for the model to understand? Some claim that a more forceful tone improves output quality. Others say that polite language like "please" actually hurts performance.

As someone trained in mathematics, my instinct is always the same: start with first principles.

What is an LLM, fundamentally?

Text vectorization. Vector similarity. Conditional probability over token sequences. Time-series memory weighting. In plain terms: a system that measures how similar the direction of two words are in high-dimensional space.

Your first prompt sets the anchor

The opening prompt defines the starting coordinate of the model's search space. My approach: establish context and framing first — let the model calibrate its position before you ask the core question.

As the conversation continues, vectors accumulate. Each exchange expands the semantic territory the model navigates. Recent exchanges carry higher memory weight — the same way you remember yesterday's dinner more clearly than a meal from three years ago.

Why does "please" reduce accuracy?

Because the model's core mechanism is vector similarity, not social convention. "Please" carries negligible semantic information — it introduces noise into the directional vector of your query, diluting the signal of what you actually want to know.

Once you understand the underlying math, these phenomena follow naturally. That said, old habits die hard. The noise stays. 🤗
